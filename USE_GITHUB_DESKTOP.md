## Step 1: Download GitHub Desktop

1. **Go to**: https://desktop.github.com
2. **Click**: "Download for Windows"

## Step 2: Install GitHub Desktop

1. **Double-click** the downloaded file (it's called `GitHubDesktopSetup.exe`)
2. **Click "Install"** or "Next" through the installation
3. **Wait** for it to install (takes 1-2 minutes)
4. **Open GitHub Desktop** when it's done

---

## Step 3: Sign In

1. **Click "Sign in to GitHub.com"**
2. **Enter your GitHub username**: `eathenbaby`
3. **Enter your password**
4. **Authorize** GitHub Desktop (click "Authorize" if asked)

---

## Step 4: Add Your Project

1. In GitHub Desktop, click **"File"** (top left)
2. Click **"Add Local Repository"**
3. Click **"Choose..."** button
4. **Navigate to**: `C:\Users\eathe\OneDrive\Desktop\Asset-Manager\Asset-Manager-cleaned\Asset-Manager`
5. **Select the folder** and click "Add repository"

## Troubleshooting

### Problem: "Repository already exists"
**Solution**: Your repository `eathenbaby/valentine` already exists on GitHub. That's fine! Just make sure GitHub Desktop is connected to it.

### Problem: Can't find the folder
**Solution**: Make sure you're selecting the `Asset-Manager` folder (the one with `package.json` inside it).

### Problem: "Publish repository" button is grayed out
**Solution**: 
1. Make sure you're signed in
2. Make sure you've selected the correct folder
3. Try refreshing GitHub Desktop
